# Mini-Projects

This repository serves as a collection of my early, beginner-level projects created using Python, HTML, CSS and JavaScript. Each project is simple and self-contained, perfect for understanding fundamental concepts.

## Projects Overview

The projects are organized into two main categories: `python` for command-line applications and `html-css-js` for web-based components and pages.

### HTML, CSS & JavaScript Projects

These projects demonstrate basic web development concepts. To view any of these, simply open the `index.html` or main `.html` file in your web browser.

| Directory | Description |
|---|---|
| `footer/` | A static footer component with social media icons and navigation links. |
| `form/` | A sample participant registration form with various input fields like text, range slider, radio buttons, and a dropdown menu for districts in Nepal. |
| `game-development/` | A static informational page about careers in game development, outlining various roles and fundamentals for beginners. |
| `home-page/` | A functional-looking home page for a mock anime streaming site named "AniSafal", featuring a search bar, an informational section, and a feedback form. |
| `navigation-bar/` | A simple, non-responsive navigation bar with basic styling. |

### Python Projects

These are simple command-line applications. To run them, you need Python installed. Navigate to the project's directory and run the script from your terminal.

**How to Run:**
```bash
python <file_name>.py
```
For example, to run the dice roller:
```bash
cd python/dices
python dices.py
```

| Directory | File | Description |
|---|---|---|
| `dices/` | `dices.py` | A simple command-line application that simulates rolling two six-sided dice. |
| `number_guesser/` | `number_guesser.py` | A classic number guessing game where the player tries to guess a random number between 1 and 100. |
| `password_generator/` | `password_generator.py` | A script that generates a random password consisting of letters, digits, and symbols with a user-defined or randomized length. |
| `rock_paper_scissors/` | `rock_paper_scissors.py` | A terminal-based Rock, Paper, Scissors game to play against the computer. |
| `running_car/` | `running_car.py` | A text-based car simulation game where you can `start`, `stop`, and `quit` using commands. |
| `weight_unit_converter/` | `weight_unit_converter.py` | A utility to convert weight between kilograms (kg) and pounds (lbs). |
| `coffee_ordering_app/` | `coffee_ordering_app.py` | A simple python program where it lets user to `order coffee`, `view order`, `checkout`, `cancel`, and `exit`. |
| `secret_language_encoder_and_decoder/` | `secret_language_encoder_and_decoder.py` | A python program where the user can encode and decode a secret language. |
